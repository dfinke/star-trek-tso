# TypeScript Web Port (Separate Directory)

This directory contains a browser-based TypeScript port of **Mainframe TSO Star Trek**.

## Run locally

1. Install dependencies:
   - `npm install`
2. Start development server:
   - `npm run dev`
3. Build production bundle:
   - `npm run build`

## Commands

- `NAV <x> <y>`
- `PHA <energy>`
- `TOR <x> <y>`
- `SRS`
- `STATUS`
- `HELP`
- `QUIT`

Use the command input in the UI to play.
